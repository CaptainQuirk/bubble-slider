# Bubble Slider

## Presentation

Un simple silder de bulle avec animation

## Installation

```
bower install
```

## Utilisation

```html
<div class="bubble-slider">
    <div class="bubble">
        Title 1
    </div>
    <div class="bubble">
        Title 2
    </div>
    <div class="bubble">
        Title 3
    </div>
</div>
```

```javascript
$('.bubble-slider').bubbleslider();
```
